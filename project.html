  • 'So You Wanna Know The Weather' is a project I began building on Christmas of 2020. It was finished it by New Year of 2021. This project was built on the stack of Python & Django. I deployed it to Heroku at first but later migrated to Vercel after an year. It gave me a new understanding of idea's & concept's behind API (Application Programming Interface) and taught me how to fetch data from varioius API's simultaneously as well as presenting them in a beautiful way. The Project was built in a timespan of a week.
  • When a user opens any website. He sents a "Request" to the Webserver. The Webserver passes the request to the Backend-Framework. In our case, the Django-Framewrok. Django then transfers the Request to our "Views" and this is where all the backend magic happens. The Views in Django are programmed to determine the Internet Protocol or I.P. Address of client from the "X-Forwarded-For" Header in the "GET" Request which includes the originiating IP Address. The IP is then passed to IpInfo API which fetches and retuns a approximate location of the user. The Location is then transferred to Tommrow's Weather API which fetches the weather for that location and returns and displays it to the user.
